## üöÄ Getting Started

### Prerequisites

- Visual Studio 2019/2022 or Visual Studio Code
- .NET Core SDK 3.1 or later
- Node.js and npm (for Angular projects)
- SQL Server (for database projects)

### Running a Project

1. Clone the repository:
```bash
git clone https://github.com/Shashank7708/swabhav.git
cd swabhav
```

2. Navigate to a specific project:
```bash
cd [ProjectName]
```

3. For C# projects:
```bash
dotnet restore
dotnet run
```

4. For Angular projects:
```bash
cd angular
npm install
ng serve
```
